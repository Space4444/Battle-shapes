# Battle Shapes

![Banner](https://raw.githubusercontent.com/Space4444/Battle-shapes/refs/heads/master/Screenshots/screen3.png)

## Introduction

This is 2D game where you control a triangular spaceship and fight against alien geometric shapes.

## [Game trailer](https://youtu.be/IeJ7vSicGfo)

## Features
- Upgrade system
- Progress saving

## Tech stack
- [Unity](https://unity.com/)

## Overview

There are infinite number of levels with increasing difficulty.
You can earn money, buy upgrades at the shop and fight against bosses.
Also enemies drop unique upgrades that you can't buy at the shop.

It is made with Unity and C# for Android.

### How to play

Open file "BattleShapes.apk" on Android. You can also open "BattleShapes.exe" on Windows,
but controls are designed only for Android.
