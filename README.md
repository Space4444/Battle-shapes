# Battle Shapes

![Banner](https://raw.githubusercontent.com/Space4444/Battle-shapes/refs/heads/master/Screenshots/screen3.png)

## Introduction

This is 2D game where you control a triangular spaceship and fight against alien geometric shapes. It is made with Unity and C#.
Versions for Android, Web and Windows are available.

## [Game trailer](https://youtu.be/IeJ7vSicGfo)

## Features
- Upgrade system
- Progress saving

- [Unity](https://unity.com/)
- [Shader Lab](https://docs.unity3d.com/Manual/SL-Reference.html)

## [Play on UnityPlay](https://play.unity.com/en/games/6987fd3c-8ecd-4d28-8f59-ca21a9befdd1/webbuild)

## Overview

There are infinite number of levels with increasing difficulty.
You can earn money, buy upgrades at the shop and fight against bosses.
Also enemies drop unique upgrades that you can't buy at the shop.

You can play on web at the links above or download and install "BattleShapes.apk" on Android or download "Win64Build" and open "BattleShapes.exe" on Windows.
